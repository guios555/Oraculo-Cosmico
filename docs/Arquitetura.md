## Objetivo

Criar uma IA conversacional temática inspirada nos Mitos de Cthulhu que varia a personalidade baseada na criatura que assumir o papel de comunicador(forma aleatória) mas ela se revelara conforme a sanidade do usuário for diminuindo.

## Componentes

- Interface gráfica
- Sistema de memória
- Banco SQLite
- Sistema de sanidade
- Motor de IA
- Arquivos Raros (Wiki)
- Diário do Pesquisador (Registro)
- Casos Abertos (Quests Interativas para progressão)
